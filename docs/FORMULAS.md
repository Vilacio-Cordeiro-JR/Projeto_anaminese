# GUIA DE FÓRMULAS E CÁLCULOS

## Índice de Massa Corporal (IMC)

### Fórmula
```
IMC = peso (kg) / altura² (m)
```

### Classificação (OMS)
| IMC | Classificação |
|-----|---------------|
| < 16.0 | Magreza grau III |
| 16.0 - 16.9 | Magreza grau II |
| 17.0 - 18.4 | Magreza grau I |
| 18.5 - 24.9 | Peso normal |
| 25.0 - 29.9 | Sobrepeso |
| 30.0 - 34.9 | Obesidade grau I |
| 35.0 - 39.9 | Obesidade grau II |
| ≥ 40.0 | Obesidade grau III |

### Limitações
- Não diferencia massa gorda de magra
- Pode classificar atletas como sobrepeso
- Não considera distribuição de gordura

## Percentual de Gordura (US Navy)

### Fórmula - Homens
```
%G = 86.010 × log₁₀(cintura - pescoço) - 70.041 × log₁₀(altura) + 36.76
```

### Fórmula - Mulheres
```
%G = 163.205 × log₁₀(cintura + quadril - pescoço) - 97.684 × log₁₀(altura) - 78.387
```

### Medidas Necessárias
- **Homens**: altura, cintura, pescoço
- **Mulheres**: altura, cintura, quadril, pescoço

### Classificação (ACE)

**Homens:**
| % Gordura | Classificação |
|-----------|---------------|
| 2-5% | Essencial |
| 6-13% | Atleta |
| 14-17% | Fitness |
| 18-24% | Aceitável |
| ≥ 25% | Obesidade |

**Mulheres:**
| % Gordura | Classificação |
|-----------|---------------|
| 10-13% | Essencial |
| 14-20% | Atleta |
| 21-24% | Fitness |
| 25-31% | Aceitável |
| ≥ 32% | Obesidade |

### Vantagens
- Método validado cientificamente
- Não invasivo
- Fácil de aplicar

## Relação Cintura-Quadril (RCQ)

### Fórmula
```
RCQ = circunferência_cintura / circunferência_quadril
```

### Classificação de Risco

**Homens:**
| RCQ | Risco |
|-----|-------|
| < 0.85 | Baixo |
| 0.85 - 0.89 | Moderado |
| ≥ 0.90 | Alto |

**Mulheres:**
| RCQ | Risco |
|-----|-------|
| < 0.75 | Baixo |
| 0.75 - 0.84 | Moderado |
| ≥ 0.85 | Alto |

### Significado
- Indica distribuição de gordura corporal
- Valores altos = gordura concentrada no abdômen
- Preditor de risco cardiovascular

---

## Relação Cintura-Altura (RCA)

### Fórmula
```
RCA = circunferência_cintura / altura
```

### Regra Prática
**"Mantenha sua cintura menor que metade da sua altura"**

### Classificação
| RCA | Classificação |
|-----|---------------|
| < 0.40 | Muito baixo (subpeso) |
| 0.40 - 0.49 | Saudável |
| 0.50 - 0.59 | Sobrepeso |
| 0.60 - 0.69 | Obesidade |
| ≥ 0.70 | Obesidade mórbida |

### Vantagens
- Melhor preditor que IMC
- Simples de calcular
- Considera estatura

## Proporções Corporais Clássicas

### Ombros e Tórax
```
Ombros ideal = Cintura × 1.6
Peitoral ideal = Cintura × 1.4
```

### Simetria Braço-Panturrilha
```
Ideal: Braço contraído ≈ Panturrilha
Razão = Braço contraído / Panturrilha
Ideal ≈ 1.0 (±5%)
```

### Proporções baseadas em Altura

| Medida | % da Altura |
|--------|-------------|
| Cintura | 45-47% |
| Peitoral | 55-60% |
| Coxa | 30-33% |
| Panturrilha | 20-22% |

### Contração Muscular
```
Ganho = (Braço contraído / Braço relaxado - 1) × 100%
Ideal: 5-15% de ganho
```

## Massas Corporais

### Massa Gorda
```
Massa Gorda (kg) = Peso × (% Gordura / 100)
```

### Massa Magra
```
Massa Magra (kg) = Peso - Massa Gorda
ou
Massa Magra (kg) = Peso × (1 - % Gordura / 100)
```

## Índice de Conicidade

### Fórmula
```
IC = (Cintura / 0.109) × √(Peso / Altura)
```

Unidades:
- Cintura: metros
- Peso: kg
- Altura: metros

### Interpretação
- Valores maiores = formato mais cônico
- Indica acúmulo central de gordura
- Complementa RCQ e RCA

## Pontuação Estética

Sistema proprietário baseado em proporções clássicas:

### Critérios (pontuação 0-100)
1. **Ombro/Cintura** (25%): proximidade de 1.6
2. **Peitoral/Cintura** (20%): proximidade de 1.4
3. **Braço/Panturrilha** (15%): proximidade de 1.0
4. **Cintura/Altura** (20%): proximidade de 45-47%
5. **Simetria Braços** (20%): ganho de 5-15%

### Classificação
| Pontuação | Classificação |
|-----------|---------------|
| 90-100 | Excepcional |
| 80-89 | Excelente |
| 70-79 | Muito Bom |
| 60-69 | Bom |
| 50-59 | Médio |
| < 50 | A Desenvolver |

## Somatotipos

### Método de Classificação

Sistema heurístico baseado em:
1. IMC
2. RCA
3. RCQ
4. Proporção ombro/cintura
5. Proporção peitoral/cintura

### Tipos

**Ectomorfo:**
- IMC baixo (< 18.5)
- RCA baixo (< 0.45)
- Proporções menores

**Mesomorfo:**
- IMC normal a alto (18.5-30)
- Boas proporções (ombro/cintura ≥ 1.4)
- Simetria equilibrada

**Endomorfo:**
- IMC alto (≥ 25)
- RCA alto (≥ 0.50)
- RCQ alto
- Cintura e quadril maiores

### Tipos Mistos
- **Ecto-Mesomorfo**: magro com potencial atlético
- **Meso-Endomorfo**: forte com tendência a ganhar gordura

## Notas Importantes

⚠️ **Limitações dos Métodos:**
- Fórmulas são estimativas, não diagnósticos
- Considerar contexto individual
- Atletas e populações especiais requerem métodos específicos
- Sempre consultar profissional qualificado

✓ **Boas Práticas:**
- Padronizar horário das medições
- Usar sempre os mesmos pontos de medida
- Calibrar equipamentos regularmente
- Documentar condições (jejum, hidratação, etc.)
